üêõ Common Issues & Solutions

"API key not valid"

  • Double-check you copied the config correctly
  • Make sure there are no extra spaces or quotes
  • Verify the project ID matches exactly

"auth/unauthorized-domain"

  • Add authorized domains in Firebase Console
  • Wait 5-10 minutes for changes to take effect
  • Make sure you're using http://localhost:8080, not file://

CORS errors

  • Must use local server (python3 -m http.server 8080)
  • Cannot use file:// protocol
  • Check authorized domains are set up
